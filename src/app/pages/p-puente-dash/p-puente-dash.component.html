Web Data Visualization Application

A dashboard application that based on information collected in the data collection application. Designed for development practitioners and researchers (In Beta).

UX Designer + Software Developer

Framing the Problem

Quickly after developing the mobile application, I found that many users desired an accompanying data visualization application to make sense of the data that was being collected. We went through multiple iterations of the design and the development to make sure we delivered a product that complemented the mobile application.

CHALLENGE

Development practitioners need a digital interface to interact with their data

Many of our clients needed a streamlined solution to manager their own data on the mobile application

SOLUTION

Dashboard Web Application

A React Application that provides automated data analysis

SCOPE

UX Design

Software Engineering

Low-Fidelity Prototyping

Balsamiq Mockup

Balsamiq is a great tool for rapid prototyping and we found it useful in helping us set the information hierarchy of our application. With a dashboard required to do so many things for different users, we were able to sketch out a couple iterations fast and settle on this one.

High-Fidelity Prototyping

Angular First Iteration

When I first learned Ionic for the mobile application, I tried to leverage my knowledge and lessons learned from that to create an Angular Application. We soon found it to be very clunky for our needs and frankly pretty bloated.

React Beta

After moving from Angular, I created a React prototype that played very well with our testers. While not finished, our users found the interface to be snappy and very modern looking. While this could've been accomplished in Angular probably, we found it easier to create and test UI kits in React.

Key Takeaways

Never stop iterating and testing

We are continually re-designing and developing this platform to match the needs of our users. The data collected from the mobile application is only as useful as the users ability to understand it and through the use of a dashboard, users are empowered to discover insights into their own data.
