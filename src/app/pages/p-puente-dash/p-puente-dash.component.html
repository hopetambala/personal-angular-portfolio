Web Data Visualization Application

A dashboard application that based on information collected in the data collection application. Designed for development practitioners and researchers (In Beta).

UX Designer + Software Developer

Framing the Problem

Quickly after developing the mobile application, I found that many users desired an accompanying data visualization application to make sense of the data that was being collected!

While still in beta, this application has been useful in distilling information to make sense to people who need efficiency.
